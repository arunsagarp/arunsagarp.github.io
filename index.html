Fourier Transform Visualization

This project visualizes the Discrete Fourier Transform by showing how complex shapes can be drawn using rotating circles (epicycles). It converts drawn paths into frequency components, then reconstructs the original drawing through animated vector addition. The visualization demonstrates how any shape can be broken down into simpler circular motions, making abstract Fourier analysis concepts visually accessible.
